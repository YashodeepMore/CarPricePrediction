About Our Model

Our car price predictor is powered by a Random Forest Regressor machine learning model. The model analyzes various car features such as mileage, engine capacity, fuel type, and transmission type to estimate the best market price.

The backend of this application is built using Flask, a lightweight web framework that allows seamless integration between the model and the user interface.
